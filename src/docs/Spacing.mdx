## CSS Variables

```css

var(--bsdex-spacing-4)      /* 4px */
var(--bsdex-spacing-8)      /* 8px */
var(--bsdex-spacing-10)     /* 10px */
var(--bsdex-spacing-12)     /* 12px */
var(--bsdex-spacing-16)     /* 16px */
var(--bsdex-spacing-24)     /* 24px */
var(--bsdex-spacing-40)     /* 40px */

/* Border Radius */
var(--bsdex-radius-sm)      /* 4px */
var(--bsdex-radius-md)      /* 8px */
var(--bsdex-radius-lg)      /* 16px */
var(--bsdex-radius-full)    /* 100px */
```
