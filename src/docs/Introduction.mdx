# BSDEX Design System

Welcome to the **BSDEX Ionic Design System** — the single source of truth for UI components, design tokens, and patterns used across BSDEX products.

## What's inside

- **Foundation** — Design tokens including colors, typography, and spacing
- **Components** — Reusable UI components with live previews and API documentation

## Getting Started

### Installation

```bash
npm install @bsdex/design-system
```

### Import tokens

```css
@import '@bsdex/design-system/tokens.css';
```

### Use components

```html
<bsdex-button type="solid" size="default">Click me</bsdex-button>
```
