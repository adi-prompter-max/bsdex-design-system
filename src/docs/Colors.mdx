# Colors

The BSDEX color palette is organized into semantic color groups. Each group contains a **base**, **tint** (lighter), and **shade** (darker) variant.

## CSS Variables

All colors are available as CSS custom properties:

```css
var(--bsdex-primary-base)    /* #6432fa */
var(--bsdex-primary-tint)    /* #7d58ff */
var(--bsdex-primary-shade)   /* #4700d8 */
var(--bsdex-success-base)    /* #289646 */
var(--bsdex-warning-base)    /* #ffc800 */
var(--bsdex-danger-base)     /* #ea0658 */
var(--bsdex-dark-base)       /* #191c1d */
var(--bsdex-medium-base)     /* #7c7c7c */
var(--bsdex-light-base)      /* #e0e3e3 */
var(--bsdex-surface)         /* #ffffff */
var(--bsdex-text)            /* #000000 */
```
