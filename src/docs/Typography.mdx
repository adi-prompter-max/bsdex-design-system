# Typography

BSDEX uses **Euclid Circular B** as the primary typeface. The type scale is designed for clarity and hierarchy across all screen sizes.

## CSS Variables

```css
/* Headings */
var(--bsdex-h1-size)           /* 28px */
var(--bsdex-h1-weight)         /* 600 */
var(--bsdex-h1-line-height)    /* 36px */

/* Body */
var(--bsdex-body-main-size)    /* 16px */
var(--bsdex-body-default-size) /* 14px */
var(--bsdex-body-small-size)   /* 12px */
var(--bsdex-body-tiny-size)    /* 10px */

/* Font Family */
var(--bsdex-font-family)       /* 'Euclid Circular B', sans-serif */
```
